### Stage 4

``` r

ensemble_fit <- ensemble_spatial(
    grm.fit.cv.1 = cmaq_fit_cv,
    grm.fit.cv.2 = modis_fit_cv,
    n.iter = 50, 
    burn = 10, 
    thin = 4,
    tau.a = 0.001, 
    tau.b = 0.001, 
    theta.tune = 0.2, 
    theta.a = 5, 
    theta.b = 0.05
)
```
